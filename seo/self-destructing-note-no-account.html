Why requiring an account defeats the purpose

If a self-destructing note service requires your email address, they have tied your identity to every note you create. A subpoena, a data breach, or a rogue employee could reveal that you created a note — and when. For sensitive communications, the metadata trail is often as revealing as the content.

SecretNote has no accounts, so there is no trail to follow.

How to send a self-destructing note right now

  1. Open SecretNote in any browser — no download, no install.
  2. Type your message, password, or secret. Everything stays in your browser until you click Create.
  3. Click Create Note. Your note is encrypted in your browser using AES-256-GCM. You get a one-time link instantly.
  4. Copy and send the link via email, chat, text — any channel. The link is safe to send because it contains only an encrypted blob ID; the decryption key lives in the URL fragment, which is never sent to our server.
  5. Your recipient opens the link once, reads the note, and it is permanently deleted. Done.

The recipient also needs no account

Reading a SecretNote requires nothing from the recipient either. They open the link in any browser, click confirm, and read. No app, no account, no sign-in. This makes SecretNote practical for sending notes to clients, external contacts, or anyone who would not want to create an account just to receive a message.

Frequently asked questions

If there is no account, how do I manage my notes?

Save the one-time link immediately after creating a note — that is the only reference to it. You can use the Delete Note Now button to destroy a note before it is read if you have the link. Without the link, there is no way to look up notes you have created, which is by design.

Is SecretNote GDPR-compliant?

SecretNote collects no personal data and stores no user profiles, which means there is minimal GDPR scope. The only personal data that could theoretically be involved is IP addresses in server logs, which are retained only for standard operational security purposes and are not linked to note content.

What if I need to send notes frequently?

The tool is designed for quick one-off use — bookmark it in your browser and use it whenever you need to share something sensitive. Each note is created in seconds. There is no workflow advantage to having an account when the tool works this fast without one.
